## Running the bot
Creating a virtual enviroment
```
python3 -m venv venv
```

Activate venv
```
source venv/bin/activate
```

Installing necessary modules
```
pip3 install -r requirements.txt
```

Run the bot
```
python3 app.py
```
Deativate venv
```
deactivate
```
